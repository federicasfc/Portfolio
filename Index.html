About Me

My name is Federica Sanchez, and I never thought I would be a coder. After I graduated college with degrees in International Studies and Linguistics, I figured my step would be grad school...or law school. However, after the pandemic hit, I started to seriously reconsider the direction that I wanted to go in. It was during this transition that I found ElevenFifty and coding. Unexpectedly I really enjoyed it, and I found a field where I was able to apply my analytical thinking skills to solve problems. As I further broaden my learning in the tech field, I would love to find a space where I can incorporate my new tech skills into the background that I have in Linguistics.

Projects

BookTracker

This project is a .NET Framework Web Application using n-tier architecture with a Blazor front-end. The app is targeted toward avid readers who need a place to track not only all the books they wish to read, but also those they have actually bought and haven't started yet.

Music API

This is a collaborative .NET Framework API Web Application using n-tier architecture. The project was built with two other developers (Michael Linn *insert Github* and Elijah Beach *insert Github*) using an Agile approach for development. This API was aimed toward music lovers who wished to keep track of their favorite artists, songs, and the labels each was associated with.

What does your name say about you?

This is a client-side app that uses JavaScript to pull from 3 different APIs to reveal information about a user's name. The user is able to input their name and find out what country they mostly likely come from, how old they likely are, and what gender they likely are.
